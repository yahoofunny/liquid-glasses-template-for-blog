# Space — Astro Glassmorphism Blog

A clean, modern blog template with a **glassmorphism** design and fullscreen video background. Built with [Astro](https://astro.build) + Tailwind CSS.

## Features

- 🔮 Glassmorphism cards with backdrop blur
- 🎥 Fullscreen video background
- 🌗 Dark theme optimized
- 📱 Responsive sidebar with mobile drawer
- 📝 Markdown/MDX posts with auto TOC
- 🚀 Deploy to Cloudflare Pages

## Quick Start

```bash
git clone https://github.com/yahoofunny/space.git
cd space
pnpm install
pnpm dev
```

## Customize

1. Edit `src/site.config.ts` — title, author, description
2. Replace `public/spacex.mp4` — your background video
3. Edit `src/pages/about.astro` — your about page
4. Add posts in `src/content/post/`

## Deploy

Push to GitHub, connect to [Cloudflare Pages](https://pages.cloudflare.com). Build command: `pnpm build`, output: `dist/`.
